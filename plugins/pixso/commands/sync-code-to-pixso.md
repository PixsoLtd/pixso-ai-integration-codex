# Sync Code to Pixso

Update or create a Pixso design from the current code implementation.

Workflow:

1. Inspect source components, tokens, and layout rules.
2. Map CSS/layout primitives to Pixso frames, auto layout, fills, strokes, effects, and text styles.
3. Preserve unsupported source properties as metadata or notes when useful.
4. Validate the resulting Pixso node structure.
